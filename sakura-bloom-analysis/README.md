# Sakura bloom analysis

## Project Description

Many tourists plan trips to Japan specifically to experience the *sakura blooming season*. However, there is always a risk of planning a trip for the wrong dates, especially now that the blooming period appears to be shifting earlier each year — a phenomenon that some associate with **global warming**.

This project analyzes **historical sakura blooming data** and environmental factors affecting flowering dates. The study also explores the relationship between bloom seasonality and tourism trends.

## Project Objective

- Investigate whether there is a **trend** toward earlier sakura blooming dates;
- Identify the **key environmental factors** affecting bloom timing;
- Determine whether **temperature** change, particularly global warming, is the **primary factor** affecting the blooming period;
- Visually evaluate whether **tourism seasonality** correlates with the sakura blooming season;
- Present the findings through an interactive analytical **dashboard** built in Power BI.
